
This new version comes with the possibility to switch between plain MQTT vs secure MQTT (over TLS / over WebSockets).
The upgrade process will:

1. look at your current configuration to determine the protocol used and save it as the settings

2. apply the proper corresponding configuration 

This new version offers a config panel, which allow to change:

 - the protocol: plain MQTT vs secure MQTT (over TLS / over WebSockets)

 - the mqtt username and/or password
